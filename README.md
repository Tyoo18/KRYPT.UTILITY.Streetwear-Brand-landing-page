## 👁️ Visual & Design Philosophy

KRYPT.UTILITY rejects corporate visual clichés. The interface draws inspiration from premium high-fashion streetwear agencies and industrial command-line interfaces.

### Design Tokens & System Assets

#### Color Palette

| Token | Hex Value | Usage |
|-------|-----------|-------|
| Pitch Black | `#080808` | Background dominance. Eliminates standard soft grays |
| Industrial Charcoal | `#121212` | Structural card containers and spatial breaks |
| Studio White | `#F5F5F7` | High-contrast editorial body text |
| Toxic Acid Volt | `#DFFF00` | Surgical high-contrast focus indicator for CTAs, countdown states, and technical feedback lines |

#### Typography Hierarchy

- **Display Headers:** Syne (Extra-bold, high-impact, tight tracking)
- **Technical Specs / UI:** Space Grotesk (Geometric, highly readable monospace hybrid)
- **Reading Copy:** Inter (Slick, minimalist sans-serif for optimal density)

#### Layout Integrity

Rigid structural borders, raw layout lines (1px border-neutral-800), strict angular elements (zero rounded corners), and transparent grid-overlays representing architectural floor plans.

## 🏗️ Technical Architecture & Stack

The prototype represents a high-performance frontend framework designed for extreme speed and instantaneous interaction feedback:
[Client / Custom Cursor] ---> [HTML5 / Tailwind UI Grid]
|
+-----------------------+-----------------------+
|                       |                       |
[Micro-Interactions]     [Interactive X-Ray]     [Asynchronous Tickers]
- Custom Pointer         - Ascii Schematics      - Live Scarcity Logs
- GSAP-like Hover States - Responsive Text Scale  - Real-time Clock loops

### Core Technologies

- **Tailwind CSS Utility Framework:** Tailored layout adjustments, custom font pairing configurations, and dynamic responsive break-points

- **Responsive Fluid Typography (Schematic Overlays):** Uses specialized responsive viewport scaling (`text-[10px] sm:text-xs md:text-[9px] lg:text-[11px] xl:text-xs 2xl:text-sm`) inside product cards to render high-density technical ASCII drawings cleanly across all screens (from mobile viewports to ultra-wide displays)

- **Micro-Interaction Suite:**
  - Interactive X-ray hover-mask on product images revealing custom ASCII structural product schematics
  - Smooth dual-axis layout custom cursor trailing system
  - Asynchronous, randomized ticker-logs to simulate high-traffic server purchasing signals (User Scarcity trigger)
  - High-precision millisecond-based system countdown loop
